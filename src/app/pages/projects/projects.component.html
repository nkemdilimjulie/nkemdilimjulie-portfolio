  • Product Review Backend Project
    • Product Review Backend

    Description: Product Review Backend project is a backend web application that enables a user to either review a product (in this case, a mobile phone) or to benefit from the list of Reviews contributed by others. The user can also link to mobile phone marketers. * A user can also use Product Review frontend App to submit reviews
    Goal: to create a secure database that stores product reviews and product marketers which channel a user to know the specific phone to buy for a specific need.
    Project Story Normally, the data of Django projects are stored automatically in dbSqlite Database. For security purpose, I was delighted to change and store data in a different Database system, like PostgreSQL, through this project.
    Project Structure and Designed Presentation: Product Review Structure
    Tech Stack: Python, Django, DRF API, PostgreSQL, Docker, Docker Compose, etc.

  • World Weather DRF API Project
    • World Weather

    Description: This is a backend Django REST Framework API that surfs current weather information from a weatherstack website * To find weather situation of any city, add ?city=CityName to my weather domain: http://127.0.0.1:8080/weather/
    Goal: World Weather project enables a user to dynamically find the weather condition (example, temperature) of any city in the world using "?city=CityName".
    Project Story I was motivated to do this project because I love API projects. An API project is just a call to consume and enjoy a finished work done by another.
    Tech Stack: Python, Django, DRF API, PostgreSQL, Docker, Docker Compose, etc.

  • Train Up A Child Project

    • * Please, Note that this project is an ongoing project

    Train Up A Child

    Description: Train-Up-A-Child is a compassionate sponsorship platform that allows users to support children's education and overall wellbeing.
    Goal: This creates a forum to enable every child - mostly, orphans and poor kids - to receive an education sponsored by kind-hearted individuals or organizations.
    Project Story I was motivated to do this project because I love children a lot. I had always wished I could sponsor the education of many kids. But I realized I could not handle it all alone. I had to give others like me, who would want to sponsor orphans' education, the opportunity to do so using this forum.
    Tech Stack: frontend - JavaScript with React (Next.js for app router), backend - Python, Django, DRF API, PostgreSQL, Docker, Docker Compose, etc.
